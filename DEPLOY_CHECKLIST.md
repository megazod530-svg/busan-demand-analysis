# 배포 체크리스트

1. busan-demand-dashboard 폴더를 GitHub 저장소로 올립니다.
2. GitHub에 node_modules, dist, .env, 엑셀 원본, 압축 파일이 올라가지 않았는지 확인합니다.
3. Vercel에서 GitHub 저장소를 Import합니다.
4. Framework Preset: Vite
5. Build Command: npm run build
6. Output Directory: dist
7. 배포 URL 접속 후 비밀번호 busan2026!로 화면을 확인합니다.
8. 외부 공유 전 링크 수신자 범위를 확인합니다.
